# barcode_app

A new Flutter project.  

프로젝트 : 택배 처리 과정 자동화, 작업 내용을 녹화하여 기록하는 프로그램 개발

목적 : 송장 바코드와 택배 바코드 자동으로 인식하고 저장하여 구성품의 일치 여부를 확인하는 기능을 구현하여 효율적인 작업 관리 시스템을 만드는 것
